\begin{solution}{exo:g3:addsub:9}
Units: $7 + \text{?} = 2$ is impossible, so $7 + 5 = 12$: the units
hole is $5$, carry $1$. Tens: $\text{?} + 2 + 1 = 6$: the tens hole
is $3$ (no carry). Hundreds: $3 + \text{?} = 8$: the hundreds hole is
$5$. So $337 + 525 = 862$, and each hole had exactly one possible
digit --- there is only one way.
\end{solution}

\begin{solution}{exo:g3:addsub:11}
