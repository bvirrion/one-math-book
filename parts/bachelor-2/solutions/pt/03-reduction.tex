\begin{solution}{exo:b2:reduction:1}

\begin{solution}{exo:b2:reduction:3}
$ X^2 - 5X + 6 = (X-2)(X-3)$: dividido com raûÙzes simples, entûÈo $ u $ ûˋ
\omterm{def:b2:reduction:diag}{diagonalizûÀvel} (\cref{cor:b2:reduction:minpolycrit}), com

\begin{solution}{exo:b2:reduction:5}

\begin{solution}{exo:b2:reduction:6}

\begin{solution}{exo:b2:reduction:11}
Trigonalizar: $ A = PTP^{-1}$, $ T = \begin{pmatrix} \lambda & c\\ 0 &
\mu\end{pmatrix}$, $\abs\lambda, \abs\mu < 1$. EntûÈo $ A^k =

\begin{solution}{pb:b2:reduction:1}
\textbf{1.} O primeiro $ k - 1$ coordenadas de $ Cv_n $ sûÈo $ u_{n+1},
\dots, u_{n+k-1}$ (os deslocamentos superdiagonais), e o û¤ltimo ûˋ
$ a_0 u_n + \dots + a_{k-1}u_{n+k-1}$. EntûÈo $ v_{n+1} = Cv_n $ detûˋm
para todos $ n $ se as û¤ltimas coordenadas corresponderem a todos $ n $, ou seja,\ se
$(\mathcal R)$ segura. Iterando, $ v_n = C^nv_0$.

coluna: as duas entradas diferentes de zero sûÈo $ X $ (posiûÏûÈo $(1,1)$) e
$-a_0$ (posiûÏûÈo $(k,1)$). O primeiro menor ûˋ $ D_{k-1}$ em forma
para os coeficientes $ a_1, \dots, a_{k-1}$; o segundo menor ûˋ
triangular superior com diagonal $-1$: determinante $(-1)^{k-1}$,
com sinal $(-1)^{k+1}$ da posiûÏûÈo. InduûÏûÈo ativada $ k $
(base $ k = 1$: $ X - a_0$) dûÀ
\[

a_0 = P(X).
\]
Para $\mu_C $: desde $ Q(C^{\mathsf T}) = Q(C)^{\mathsf T}$ para qualquer
polinûÇmio, $ C $ e $ C^{\mathsf T}$ tem o mesmo \omterm{def:b2:linalg:annihilator}{aniquiladores},
daûÙ o mesmo polinûÇmio mûÙnimo. Para $ C^{\mathsf T}$: o
colunas lidas $ C^{\mathsf T}e_1 = e_2$, \dots, $ C^{\mathsf
T}e_{k-1} = e_k $, entûÈo $(e_1, C^{\mathsf T}e_1, \dots, (C^{\mathsf
T})^{k-1}e_1)$ ûˋ a base canûÇnica: grûÀtis. Um polinûÇmio $ Q
\neq 0$ de grau $< k $ entûÈo tem $ Q(C^{\mathsf T})e_1 \neq 0$
(ûˋ uma combinaûÏûÈo nûÈo trivial de vetores de base): $\deg\mu \geq
k $. Como $\mu \mid \chi = P $ com $\deg P = k $: $\mu_C = P $.

\textbf{3.} Para $ v = (1, \lambda, \dots, \lambda^{k-1})^{\mathsf
T}$: linhas $1$ para $ k-1$ de $ Cv $ dar $\lambda, \lambda^2, \dots,
\lambda^{k-1}$, ou seja\ $\lambda $ vezes o primeiro $ k - 1$ entradas
de $ v $; a û¤ltima linha dûÀ $\sum_m a_m\lambda^m = \lambda^k -
P(\lambda) = \lambda^k = \lambda\cdot\lambda^{k-1}$. EntûÈo $ Cv =
\lambda v $. Por outro lado, as equaûÏûçes $(Cx)_i = \lambda x_i $ para
$ i < k $ ler $ x_{i+1} = \lambda x_i $: qualquer vetor prû°prio ûˋ
proporcional a $ v $ --- todo autoespaûÏo tem dimensûÈo exatamente
$1$. \omterm{def:b2:reduction:diag}{DiagonalizûÀvel} se o autoespaûÏo dimensûçes somam a $ k $
(\cref{thm:b2:reduction:diagcrit}) se houver $ k $ distinto
valores prû°prios se $ P $ tem $ k $ raûÙzes distintas (a valores prû°prios sûÈo
as raûÙzes de $\chi_C = P $).

\textbf{4.} Cada $(\lambda_i^n)_n $ resolve $(\mathcal R)$:
$\lambda_i^{n+k} = \lambda_i^n\,\lambda_i^k =
\lambda_i^n\sum_m a_m\lambda_i^m $. Liberdade: um desaparecimento
combinaûÏûÈo $\sum_i c_i\lambda_i^n = 0$ para $ n = 0, \dots, k-1$
ûˋ um sistema Vandermonde (\cref{exo:b2:linalg:11}) no $ c_i $:

\textbf{5.} $ P = X^2 - X - 6 = (X - 3)(X + 2)$: soluûÏûÈo geral

\textbf{7.} A prova de \cref{thm:b2:reduction:kernels} usa

\textbf{8.} Para $ Q \in \C[X]$: $(S -

\textbf{9.} Combine as questûçes 7 e 8: toda soluûÏûÈo se decompûçe

\textbf{10.} $ P = X^2 - 4X + 4 = (X - 2)^2$: soluûÏûçes $(a +

\textbf{12.} Calcular:
\[
q(a_{n+1}, b_{n+1}) = (a_n + 2b_n)^2 - 2(a_n + b_n)^2
= -a_n^2 + 2b_n^2 = -q(a_n, b_n).

Com $ q(a_0, b_0) = 1 - 2 = -1$: $ a_n^2 - 2b_n^2 = (-1)^{n+1}$.
Estruturalmente: $ q(a, b) = (a - \sqrt2\,b)(a + \sqrt2\,b)$ e
o mapa linear $ M $ multiplica o fator $ a + \sqrt2 b $ por $1 +
\sqrt2$ e o fator $ a - \sqrt2 b $ por $1 - \sqrt2$ (calcular:
$ a_{n+1} + \sqrt2 b_{n+1} = (1 + \sqrt2)(a_n + \sqrt2 b_n)$);
o produto ûˋ multiplicado por $(1 + \sqrt2)(1 - \sqrt2) = -1 =
\det M $ cada passo.

\textbf{13.} Desde $ a_n^2 - 2b_n^2 = (a_n - \sqrt2 b_n)(a_n +

\[

= \frac{\abs{a_n^2 - 2b_n^2}}{b_n(a_n + \sqrt2 b_n)}
= \frac{1}{b_n(a_n + \sqrt2 b_n)} \leq \frac1{2b_n^2},
