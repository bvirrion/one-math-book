\begin{solution}{exo:b1:complex:10}
Sea $j = \eu^{2\iu\pi/3}$, entonces $j^2 + j + 1 = 0$ y $\eu^{\iu\pi/3} =
-j^2$,$\eu^{-\iu\pi/3} = -j$. El triangulo es equilatero directo si y solo
$c - a = -j^2 (b - a)$, si indirecta $c - a = -j(b - a)$
(\cref{ex:b1:complex:equilateral}).

Considere $P = a + jb + j^2 c$ y $Q = a + j^2 b + jc $. Usando $1 +
j^2 = -j $ y $j^3 = 1$:
\[
c - a + j^2(b - a) = c + j^2 b - (1 + j^2)\,a = c + j^2 b + ja
= j\,(a + jb + j^2 c) = jP,
\]
entonces el caso directo dice $jP = 0 \iff P = 0$; el mismo cĂĄlculo
con $j$ en lugar de $j^2$ da $c - a + j(b - a) = j^2 Q$, por lo que el
el caso indirecto dice $Q = 0$. Por lo tanto: equilĂĄtero (cualquier orientaciĂłn)
$\iff PQ = 0$. Ampliando, con $j + j^2 = -1$:
\[
PQ = a^2 + b^2 + c^2 + (j + j^2)(ab + bc + ca)
= a^2 + b^2 + c^2 - (ab + bc + ca).
\]
Entonces el triĂĄngulo es equilĂĄtero si y sĂłlo si $a^2 + b^2 + c^2 = ab +
bc + ca $.
\end{solution}

\begin{solution}{exo:b1:complex:11}
Ya que $\omega^k$,$k = 0, \dots, n-1$, son exactamente las raĂ­ces $n$
de $X^n - 1$(\cref{thm:b1:complex:roots}), y el polinomio es

= (X - 1) \prod_{k=1}^{n-1} (X - \omega^k).
\]
Dividiendo por $X - 1$:$\;1 + X + \dots + X^{n-1} = \prod_{k=1}^{n-1}
(X - \omega^k)$. La evaluaciĂłn en $X = 1$ da $P = n $.

Ahora $1 - \omega^k = -\eu^{\iu k\pi/n}\bigl(\eu^{\iu k\pi/n} -
\eu^{-\iu k\pi/n}\bigr) = -2\iu\,\eu^{\iu k\pi/n} \sin\frac{k\pi}{n}$,

= 2^{n-1} \prod_{k=1}^{n-1} \sin\frac{k\pi}{n},
\qquad\text{hence}\qquad

\end{solution}
