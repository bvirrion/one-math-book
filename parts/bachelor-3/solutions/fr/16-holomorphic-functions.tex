\section*{Chapitre \ref{ch:b3:holomorphic} --- Fonctions
holomorphes}

\begin{solution}{exo:b3:holomorphic:1}
(a) $\bar z$~: $P = x$, $Q = -y$, donc $P_x = 1 \neq -1 =
Q_y$~: nulle part $\C$-différentiable. $\abs z^2$~: $P = x^2
+ y^2$, $Q = 0$~: Cauchy--Riemann exige $2x = 0 = 2y$~:
différentiable en $0$ seulement --- et \omterm{def:b3:holomorphic:holo}{holomorphe} nulle part
(pas d'\omterm{def:b3:topology:topology}{ouvert}). $\operatorname{Re}z$~: $P_x = 1 \neq 0 =
Q_y$~: nulle part.

(b) $x^2 - y^2 = \operatorname{Re}(z^2)$~: $f(z) = z^2$
convient. Toutes les solutions~: si $\operatorname{Re}f =
\operatorname{Re}g$ avec $f, g$ \omterm{def:b3:holomorphic:holo}{holomorphes} sur le \omterm{def:b3:topology:connected}{connexe}
$\C$, alors $h = f - g$ a $\operatorname{Re}h = 0$~;
Cauchy--Riemann donne $h' = P_x + \iu Q_x = 0 - \iu P_y =
0$~: $h$ est une constante imaginaire. Réponse~: $f(z) =
z^2 + \iu c$, $c \in \R$.
\end{solution}

\begin{solution}{exo:b3:holomorphic:2}
Sur le cercle unité $\gamma(t) = \eu^{\iu t}$~:
\[
\int_C z^n\,\dd z = \int_0^{2\pi}\eu^{\iu nt}\,\iu\eu^{\iu
t}\dd t = \iu\int_0^{2\pi}\eu^{\iu(n+1)t}\dd t
= \begin{cases} 2\iu\pi & n = -1,\\ 0 & n \neq -1.\end{cases}
\]
Pour $\bar z$~: le long de $[0, 1+\iu]$, $\gamma(t) = t(1 +
\iu)$~: $\int_0^1 t(1 - \iu)(1 + \iu)\dd t = \int_0^12t\,\dd
t = 1$. Le long de $0 \to 1 \to 1 + \iu$~: $\int_0^1t\dd t +
\int_0^1(1 - \iu t)\,\iu\,\dd t = \frac12 + \iu + \frac12 = 1
+ \iu$. Valeurs différentes entre les mêmes extrémités~: par
la~\cref{prop:b3:holomorphic:primitive}, $\bar z$ n'a pas de
primitive sur aucun \omterm{def:b3:topology:topology}{ouvert} contenant les deux chemins.
\end{solution}

\begin{solution}{exo:b3:holomorphic:3}
(a) Le plan découpé $\Omega = \C\setminus\intoc{-\infty}0$
est étoilé par rapport à $1$, et $\frac1z \in \mathcal
H(\Omega)$~: le~\cref{thm:b3:holomorphic:cauchy} fournit une
primitive $L$ avec $L(1) = 0$. Alors
$\bigl(z\eu^{-L(z)}\bigr)' = \eu^{-L}(1 - z\cdot\frac1z) =
0$~: $z = c\,\eu^{L(z)}$ avec $c = 1$ (en $z = 1$). Écrivant
$L = u + \iu v$~: $\abs z = \eu^u$ et $z = \abs z\eu^{\iu
v}$ avec $v$ \omterm{def:b3:topology:continuity}{continue}, $v(1) = 0$, $v \in \intoo{-\pi}\pi$
($v$ est un argument continu de $z$ sur le \omterm{def:b3:topology:connected}{connexe} $\Omega$,
donc son image évite les multiples impairs de $\pi$ ---
aucun point de $\Omega$ n'est sur $\R_-$ --- et, contenant
$v(1) = 0$, reste dans $\intoo{-\pi}\pi$~: $v$ est
l'argument principal)~: $L = \log$.

(b) Si $g$ était un logarithme continu sur $\C^*$~: $h(t) =
g(\eu^{\iu t})$ satisfait $\eu^{h(t)} = \eu^{\iu t}$, donc
$h(t) - \iu t \in 2\iu\pi\Z$, et par continuité $h(t) = \iu
t + 2\iu\pi k$ pour un entier $k$ fixe. Alors $g(1) = h(0) =
2\iu\pi k$ et $g(1) = h(2\pi) = 2\iu\pi(k + 1)$~:
contradiction.

(c) Sur $D(0,1)$~: $\log(1 + z) =
\sum_{n\geq1}\frac{(-1)^{n+1}}{n}z^n$ --- les deux membres
s'annulent en $0$ et ont pour dérivée $\frac1{1+z} =
\sum(-1)^nz^n$ (l'\cref{ex:b3:holomorphic:examples})~; une
primitive est unique à constante près sur le disque
\omterm{def:b3:topology:connected}{connexe}.
\end{solution}

\begin{solution}{exo:b3:holomorphic:4}
(a) Développer en $0$ (rayon $\infty$)~: par les \omterm{thm:b3:holomorphic:analytic}{estimations
de Cauchy} sur $C_r$, $\abs{c_k} \leq C(1 + r)^n/r^k \to 0$
quand $r \to \infty$ pour $k > n$~: $f = \sum_{k\leq
n}c_kz^k$.

(b) Si $\operatorname{Re}f \leq M$~: $g = \eu^f$ est entière
avec $\abs g = \eu^{\operatorname{Re}f} \leq \eu^M$~:
constante par Liouville. Alors $g' = f'g = 0$ avec $g$ non
nulle~: $f' = 0$, et $f$ est constante.

(c) Si $f$ omet le demi-plan $H$, une application affine $w
\mapsto \alpha w + \beta$ envoie $\C\setminus H$ dans
$\{\operatorname{Re} \leq M\}$~; appliquer (b) à $\alpha f +
\beta$.
\end{solution}

\begin{solution}{exo:b3:holomorphic:5}
(a) $g(z) = f(z) - z^2$ s'annule aux points $\frac1n$, qui
s'accumulent en $0 \in \Omega$~: par le théorème d'identité
($\Omega$ \omterm{def:b3:topology:connected}{connexe}), $g \equiv 0$~: $f(z) = z^2$.

(b) Oui~: $f(z) = z\cos(\pi/z)$ est \omterm{def:b3:holomorphic:holo}{holomorphe} sur $\C^*$
(composition) et $f(\frac1n) = \frac1n\cos(n\pi) =
\frac{(-1)^n}n$. Pas de contradiction avec (a)~: le point
d'accumulation $0$ des nœuds d'interpolation n'appartient
\emph{pas} à $\C^*$, donc le théorème d'identité se tait ---
deux fonctions distinctes ($z\cos(\pi/z)$ et, disons, celle
d'une autre interpolation) peuvent partager ces valeurs.

(c) $f \equiv 0$ partout, versus $g = 0$ sur $D(0,1)$ et $g =
1$ sur $D(3,1)$~: \omterm{def:b3:holomorphic:holo}{holomorphes} sur l'union non \omterm{def:b3:topology:connected}{connexe}, égales
sur $D(0,1)$, différentes. L'argument ouvert-fermé du
théorème d'identité a besoin de la connexité pour propager
d'une composante à l'autre --- et ne le peut pas.
\end{solution}

\begin{solution}{exo:b3:holomorphic:6}
(a) Par le principe du maximum appliqué sur le domaine
borné~: $\sup_{\mathbb D}\abs f = \sup_{\partial\mathbb
D}\abs f = 1$. Comme $f$ n'a pas de zéro, $1/f$ est
\omterm{def:b3:holomorphic:holo}{holomorphe} sur $\mathbb D$, \omterm{def:b3:topology:continuity}{continue} sur l'\omterm{def:b3:topology:interior}{adhérence}, de
\omterm{def:b3:modules:module}{module} bord $1$~: de même $\abs{1/f} \leq 1$, c'est-à-dire
$\abs f \geq 1$. Donc $\abs f \equiv 1$~: le \omterm{def:b3:modules:module}{module} atteint
un maximum intérieur, et le~\cref{thm:b3:holomorphic:maximum}(2)
force $f$ constante.

(b) $f(z) = z$~: \omterm{def:b3:modules:module}{module} bord $1$, zéro à l'origine,
non constante --- le zéro bloque exactement l'argument
$1/f$.
\end{solution}

\begin{solution}{exo:b3:holomorphic:7}
Continuité de $G$~: convergence dominée avec dominateur $h$
(borne localement uniforme). Holomorphie par Morera (établi
dans le~\cref{thm:b3:holomorphic:weierstrassconv})~: pour un
triangle fermé $T$ dans un disque où $\abs g \leq h$,
\[
\int_{\partial T}G(z)\,\dd z =
\int_X\Bigl(\int_{\partial T}g(x, z)\,\dd z\Bigr)\dd\mu(x) =
0,
\]
l'interversion par Fubini ($\int_X\int_{\partial T}\abs g
\leq \operatorname{length}(\partial T)\int h < \infty$) et
l'annulation intérieure par Goursat. Pour $\Gamma$~: sur la
bande $a \leq \operatorname{Re}z \leq b$ ($0 < a \leq b$),
$\abs{t^{z-1}\eu^{-t}} = t^{\operatorname{Re}z-1}\eu^{-t}
\leq (t^{a-1} + t^{b-1})\eu^{-t}$, \omterm{def:b3:lebesgue:l1}{intégrable} sur
$\intoo0{+\infty}$~: $\Gamma$ est \omterm{def:b3:holomorphic:holo}{holomorphe} sur
$\{\operatorname{Re} z > 0\}$ (la \omterm{def:b3:measure:measure}{mesure} n'est que
$\sigma$-finie, mais l'argument n'a besoin que du
dominateur \omterm{def:b3:lebesgue:l1}{intégrable}). Par le théorème d'identité,
l'équation fonctionnelle $\Gamma(z + 1) = z\Gamma(z)$,
prouvée sur $\intoo0{+\infty}$
(l'\cref{ex:b3:lebesgue:gamma}), vaut sur tout le demi-plan.
\end{solution}

\begin{solution}{exo:b3:holomorphic:8}
Écrire $P = c\prod_k(X - a_k)^{m_k}$
(le~\cref{pb:b3:holomorphic:1}). Soit $P'(z) = 0$. Si $P(z)
= 0$, alors $z$ est l'un des $a_k$~: dans l'enveloppe. Sinon,
la dérivée logarithmique donne
\[
0 = \frac{P'(z)}{P(z)} = \sum_k\frac{m_k}{z - a_k}
= \sum_k m_k\,\frac{\bar z - \bar a_k}{\abs{z - a_k}^2} ;
\]
en conjuguant, $\sum_kw_k(z - a_k) = 0$ avec $w_k =
m_k/\abs{z - a_k}^2 > 0$~: $z = \sum_k\frac{w_k}{\sum
w}\,a_k$, une combinaison convexe des racines. Pour $P =
z^3 - 1$~: racines les racines cubiques de l'unité, $P' =
3z^2$ avec racine double $0$ --- le barycentre du triangle
équilatéral.
\end{solution}

\begin{solution}{exo:b3:holomorphic:9}
Le carré unité fermé $K = \{x + \iu y : 0 \leq x, y \leq 1\}$
est compact~: $M = \sup_K\abs f < \infty$. Tout $z \in \C$
diffère d'un point de $K$ d'un élément de $\Z + \iu\Z$
(soustraire les parties entières), et $f$ est invariante sous
ces translations (itérer les deux relations)~: $\abs f \leq
M$ sur $\C$. Liouville~: $f$ est constante. D'où toute
fonction méromorphe doublement périodique non constante ---
les fonctions elliptiques de la théorie classique --- doit
avoir des pôles.
\end{solution}

\begin{solution}{exo:b3:holomorphic:10}
(a) Prendre les parties réelles dans la formule de la moyenne
(le~\cref{thm:b3:holomorphic:maximum}(1)). Régularité~: $f$
est analytique, donc $P, Q \in \mathcal C^\infty$~; dériver
Cauchy--Riemann~: $P_{xx} = (Q_y)_x = (Q_x)_y = (-P_y)_y =
-P_{yy}$ (symétrie de Schwarz des dérivées secondes)~:
$\Delta P = 0$.

(b) Si $\operatorname{Re}f$ atteignait un maximum intérieur
sur un $\Omega$ \omterm{def:b3:topology:connected}{connexe}~: $g = \eu^f$ a $\abs g =
\eu^{\operatorname{Re}f}$ atteignant un maximum intérieur,
donc $g$, d'où $\operatorname{Re}f = \ln\abs g$, est
constante (le~\cref{thm:b3:holomorphic:maximum}(2)). Sur un
domaine borné avec continuité jusqu'au bord,
$\sup_{\bar\Omega}\operatorname{Re}f =
\sup_{\partial\Omega}\operatorname{Re}f$.
\end{solution}

\begin{solution}{exo:b3:holomorphic:11}
(a) Les deux formules coïncident sur $I$ ($z = \bar z$ et $f$
réelle là~: $\overline{f(\bar z)} = \overline{f(z)} =
f(z)$), et $z \mapsto \overline{f(\bar z)}$ est \omterm{def:b3:topology:continuity}{continue} sur
$\Omega^-\cup I$ comme composition d'\omterm{def:b3:topology:continuity}{applications
continues}~: $F$ est \omterm{def:b3:topology:continuity}{continue} sur $\Omega$. Holomorphie sur
$\Omega^-$~: près de $z_0 \in \Omega^-$, développer $f(w) =
\sum c_n(w - \bar z_0)^n$ près de $\bar z_0 \in \Omega^+$~;
alors
\[
\overline{f(\bar z)} = \sum_n\bar c_n\,(z - z_0)^n,
\]
une série entière convergente~: \omterm{def:b3:holomorphic:holo}{holomorphe}.

(b) Les triangles évitant $I$ sont traités par Goursat dans
$\Omega^\pm$. Pour un triangle rencontrant $I$, le couper
par l'axe réel en au plus trois triangles/quadrilatères,
chacun avec un côté sur $I$~; pour un tel morceau $P$ \omterm{def:b3:rings:content}{contenu}
dans, disons, $\overline{\Omega^+}$, l'\omterm{def:b3:holomorphic:contour}{intégrale de contour}
est la limite quand $\varepsilon \to 0^+$ de l'intégrale sur
des translatées de type $P + \iu\varepsilon$ (la continuité
uniforme de $F$ sur le morceau compact fait converger les
intégrales de bord, le côté sur $I$ étant approché par le
haut), et chaque translatée est dans $\Omega^+$ où Goursat
donne $0$. En sommant les morceaux~: $\int_{\partial T}F =
0$. Morera (le critère dans
le~\cref{thm:b3:holomorphic:weierstrassconv})~: $F$ est
\omterm{def:b3:holomorphic:holo}{holomorphe} sur $\Omega$.

(c) Sur le disque, $G(z) = \overline{f(\bar z)}$ est
\omterm{def:b3:holomorphic:holo}{holomorphe} par le calcul de (a) et coïncide avec $f$ sur le
diamètre, ensemble à points d'accumulation~: $G = f$
partout (théorème d'identité). Si $f$ était réelle sur un
\omterm{def:b3:topology:topology}{ouvert} non vide $U$~: sur $U$ les deux dérivées partielles de
$Q = \operatorname{Im}f$ s'annulent, et Cauchy--Riemann
transfère cela à $P = \operatorname{Re}f$ ($P_x = Q_y = 0$,
$P_y = -Q_x = 0$), donc $f' = P_x + \iu Q_x = 0$ sur $U$~:
$f$ est constante sur $U$, d'où partout par le théorème
d'identité ($\Omega$ \omterm{def:b3:topology:connected}{connexe}).
\end{solution}

\begin{solution}{exo:b3:holomorphic:12}
(a) $1 = f^2 + g^2 = (f + \iu g)(f - \iu g)$, donc $h = f +
\iu g$ ne s'annule jamais (son cofacteur devrait exploser).
Pour $h$ entière sans zéro~: $h'/h$ est entière, et $\C$ est
étoilé, donc elle a une primitive $\varphi_0$
(le~\cref{thm:b3:holomorphic:cauchy})~; alors
$\bigl(h\eu^{-\varphi_0}\bigr)' = \eu^{-\varphi_0}(h' -
h\varphi_0') = 0$~: $h = c\,\eu^{\varphi_0}$ avec $c \neq
0$, et en absorbant une constante $\log c$ dans $\varphi =
\varphi_0 + \log c$ (n'importe quel logarithme complexe de
$c$)~: $h = \eu^{\varphi}$.

(b) Avec $h = \eu^{\varphi}$ et $h^{-1} = f - \iu g =
\eu^{-\varphi}$~:
\[
f = \frac{\eu^{\varphi} + \eu^{-\varphi}}2,
\qquad
g = \frac{\eu^{\varphi} - \eu^{-\varphi}}{2\iu} .
\]
En écrivant $\varphi = \iu\psi$ avec $\psi = -\iu\varphi$
entière, cela lit $f = \cos\psi$, $g = \sin\psi$~: les
solutions entières sont exactement les couples $(\cos\psi,
\sin\psi)$ avec $\psi$ entière, et la réciproque est
l'identité $\cos^2 + \sin^2 = 1$.
Pour $f^2 + g^2 = 0$~: $(f + \iu g)(f - \iu g) = 0$ dans
l'anneau intègre $\mathcal H(\C)$ ($\C$ \omterm{def:b3:topology:connected}{connexe}~: des
diviseurs de zéro violeraient le théorème d'identité)~: $g =
\pm\iu f$ avec $f$ entière arbitraire.
\end{solution}

\begin{solution}{pb:b3:holomorphic:1}
\textbf{1.} Pour $\abs z \geq 1$~:
\[
\abs{P(z)} \geq \abs z^n\Bigl(1 -
\frac{\abs{a_{n-1}}}{\abs z} - \dots -
\frac{\abs{a_0}}{\abs z^n}\Bigr)
\geq \abs z^n\Bigl(1 - \frac{A}{\abs z}\Bigr),
\qquad A = \sum_k\abs{a_k} :
\]
pour $\abs z \geq R_0 = \max(1, 2A)$, $\abs{P(z)} \geq
\frac12\abs z^n \to \infty$.

\textbf{2.} Choisir $R \geq R_0$ avec $\frac12R^n \geq
\abs{P(0)}$. Sur le compact $\bar D(0, R)$ le continu $\abs
P$ atteint un minimum, en un certain $z_0$~; à l'extérieur,
$\abs P \geq \frac12R^n \geq \abs{P(0)} \geq \abs{P(z_0)}$~:
le minimum est global.

\textbf{3.} $Q(h) = P(z_0 + h)/P(z_0)$ est un polynôme en $h$
avec $Q(0) = 1$~; il est non constant ($P$ l'est), donc un
coefficient au-delà de la constante est non nul~: $Q(h) = 1
+ c_kh^k + h^{k+1}S(h)$ avec $k \geq 1$ minimal, $c_k \neq
0$, $S \in \C[X]$.

\textbf{4.} Racines~: tout $w = \rho\eu^{\iu\varphi} \neq 0$
a la racine $k$-ième $\rho^{1/k}\eu^{\iu\varphi/k}$, où
$\rho^{1/k}$ existe par le théorème des valeurs
intermédiaires appliqué à $t \mapsto t^k$ sur
$\intco0\infty$ --- pas de circularité. Choisir $\omega$
avec $\omega^k = -1/c_k$. Alors
\[
Q(t\omega) = 1 - t^k + t^{k+1}\,\omega^{k+1}S(t\omega),
\qquad
\abs{Q(t\omega)} \leq 1 - t^k + C\,t^{k+1}
\quad (0 < t \leq 1),
\]
avec $C = \abs\omega^{k+1}\sup_{\abs h \leq
\abs\omega}\abs{S(h)}$ (noter $1 - t^k \geq 0$ sur
$\intcc01$).

\textbf{5.} Pour $0 < t < \min(1, 1/C)$~: $\abs{Q(t\omega)}
\leq 1 - t^k(1 - Ct) < 1$, c'est-à-dire $\abs{P(z_0 +
t\omega)} < \abs{P(z_0)}$ --- contredisant la minimalité
globale. Donc $P(z_0) = 0$~: la preuve de d'Alembert--Argand
est complète.

\textbf{6.} Version Liouville~: si $P$ ne s'annule jamais,
$1/P$ est entière~; par la question~1, $\abs{1/P} \leq
2R_0^{-n}$ hors de $\bar D(0, R_0)$, et $1/P$ est \omterm{def:b3:topology:continuity}{continue}
sur ce disque compact, donc bornée là aussi~: entière
bornée, donc constante (le~\cref{cor:b3:holomorphic:liouville}),
rendant $P$ constant~: absurde. Ingrédients~: d'Alembert
utilise la compacité (existence du minimum) et l'existence
via forme polaire des racines $k$-ièmes~; Liouville utilise
tout l'appareil de Cauchy (Goursat --- lui-même un argument
de compacts emboîtés --- et les \omterm{thm:b3:holomorphic:analytic}{estimations de Cauchy}) plus
la même coercivité. La compacité des disques fermés est le
noyau commun, irréductible.

\textbf{7.} Si $\deg P \geq 1$, prendre une racine $\alpha$
(questions~5)~; diviser~: $P = (X - \alpha)Q + P(\alpha) =
(X - \alpha)Q$, avec $\deg Q = n - 1$~; récurrence. En
regroupant les facteurs égaux~: $P = c\prod_i(X -
\alpha_i)^{m_i}$, $\sum m_i = n$.

\textbf{8.} Pour $P$ réelle~: $P(\bar\alpha) =
\overline{P(\alpha)} = 0$, et les multiplicités s'accordent
(conjuguer la factorisation)~: les racines non réelles
viennent par paires, contribuant $(X - \alpha)(X -
\bar\alpha) = X^2 - 2\operatorname{Re}(\alpha)X +
\abs\alpha^2$, un quadratique réel de discriminant $< 0$.
D'où la liste d'irréductibles énoncée, et un polynôme réel
de \omterm{def:b3:galois:extension}{degré} impair, ayant un nombre pair de racines non
réelles, doit en avoir une réelle. Preuve directe~: $P(x)
\to \pm\infty$ quand $x \to \pm\infty$ (\omterm{def:b3:galois:extension}{degré} impair,
coefficient dominant positif disons), donc $P$ change de
signe, et le théorème des valeurs intermédiaires
s'applique. Pour $X^3 - X - 1$~: les deux arguments donnent
l'unique racine réelle $\approx 1{,}3247$ (et une paire
conjuguée).

\textbf{9.} (i) $\chi_u \in \C[X]$ est non constant~: il a
une racine $\lambda$, et $\det(u - \lambda\,\mathrm{id}) =
0$ donne un vecteur propre~; la machinerie des \omterm{thm:b3:modules:structure}{diviseurs
élémentaires} du~\cref{thm:b3:modules:jordan} s'applique
alors à toute matrice complexe, $\chi$ se scindant
toujours. (ii) Soit $P \in \bar\Q[X]$ non constant. Comme
polynôme sur $\C$ il a une racine $z \in \C$~; $z$ est
\omterm{def:b3:galois:algebraic}{algébrique} sur $\bar\Q$, d'où sur $\Q$ par transitivité
(le~\cref{cor:b3:galois:algclosed}), donc $z \in \bar\Q$~:
tout polynôme non constant sur $\bar\Q$ a une racine dans
$\bar\Q$.

\textbf{10.} Sur $\Q(\iu)$, la question~4 échoue déjà~: les
racines $k$-ièmes n'existent pas forcément (pas de
$\sqrt2$), et même en les accordant, la question~2 échoue
--- une suite minimisante n'a pas à converger, $\Q$ manquant
de complétude~; dans la route de Liouville, les triangles
compacts emboîtés de Goursat ont une intersection vide sur
les points de $\Q(\iu)$. Les deux preuves consomment la
complétude (équivalemment, via la convergence monotone
bornée, la complétude d'ordre) de $\R$~; la connexité
alimente le théorème des valeurs intermédiaires derrière la
forme polaire. L'énoncé «~$\C$ est \omterm{def:b3:galois:closure}{algébriquement clos}~»
est de l'algèbre~; toute preuve connue en est de l'analyse
introduite en contrebande par la définition de $\R$.

\textbf{11.} Par le~\cref{thm:b3:holomorphic:analytic} en
$a$, $f^{(n)}(a) = n!\,c_n$ avec
\[
c_n = \frac1{2\iu\pi}\int_{C_r}\frac{f(w)}{(w -
a)^{n+1}}\,\dd w,
\qquad
\abs{c_n} \leq \frac{2\pi r}{2\pi}\cdot
\frac{\sup_{C_r}\abs f}{r^{n+1}}
= \frac{\sup_{C_r}\abs f}{r^{n}} :
\]
les \omterm{thm:b3:holomorphic:analytic}{estimations de Cauchy}, sous la forme affichée après
multiplication par $n!$. Si $\abs f \leq M$ sur $\C$~: pour
tout $a$ et tout $r$, $\abs{f'(a)} \leq M/r \to 0$ quand $r
\to \infty$, donc $f' \equiv 0$ et $f$ est constante sur le
\omterm{def:b3:topology:connected}{connexe} $\C$ --- Liouville retrouvé.

\textbf{12.} Développer $f = \sum_kc_kz^k$ en $0$ (rayon
$\infty$). Pour $k > m$~: $\abs{c_k} \leq (A + Br^m)/r^k \to
0$ quand $r \to \infty$, donc $c_k = 0$~: $f = \sum_{k\leq
m}c_kz^k$ est un polynôme de \omterm{def:b3:galois:extension}{degré} au plus $m$.

\textbf{13.} $g = \eu^f$ est entière avec $\abs g =
\eu^{\operatorname{Re}f} \leq \eu^M$~: constante par la
question~11. Alors $0 = g' = f'g$ avec $g$ sans zéro~: $f' =
0$ et $f$ est constante.

\textbf{14.} Soit $M = \sup_K\abs f$ sur le carré unité fermé
compact $K$. Tout $z$ diffère d'un point de $K$ d'un élément
de $\Z + \iu\Z$ (soustraire les parties entières), et
itérer les deux relations de périodicité laisse $f$
inchangée~: $\abs f \leq M$ sur tout $\C$, et la question~11
rend $f$ constante. Donc une fonction non constante
invariante sous le réseau ne peut être entière~: les
fonctions elliptiques de la théorie classique doivent
porter des pôles --- la porte historique
du~\cref{ch:b3:residues}.

\textbf{15.} Si $f(\C)$ manque le disque $D(a, r)$, alors
$\abs{f(z) - a} \geq r$ pour tout $z$, donc $g = 1/(f - a)$
est entière avec $\abs g \leq 1/r$~: constante par
Liouville, d'où $f$ constante. Contraposée~: l'image d'une
fonction entière non constante rencontre tout disque ---
elle est \omterm{def:b3:topology:interior}{dense} dans $\C$.

\textbf{16.} Choisir $R$ avec $\abs f \geq 1$ hors de $D(0,
R)$. Les zéros de $f$ sont dans le compact $\bar D(0, R)$~;
s'ils étaient infinis, ils s'y accumuleraient, et
le~\cref{thm:b3:holomorphic:identity} forcerait $f \equiv 0$
--- impossible. Les appeler $z_1, \dots, z_p$, avec
multiplicités $m_1, \dots, m_p$, poser $M = \sum m_i$ et
$\Pi(z) = \prod_i(z - z_i)^{m_i}$. En factorisant chaque
zéro hors de la série entière, $g = f/\Pi$ est entière et
sans zéro. Pour $\abs z \geq \max(R, 2\max_i\abs{z_i})$~:
$\abs{z - z_i} \leq 2\abs z$ et $\abs f \geq 1$, donc
$\abs{1/g} = \abs\Pi/\abs f \leq 2^M\abs z^M$~; sur le
disque compact restant $1/g$ est \omterm{def:b3:topology:continuity}{continue}, donc bornée~:
$\abs{1/g} \leq A + B\abs z^M$ partout. Par la question~12,
$1/g$ est un polynôme~; sans zéro, donc par la question~7
c'est une constante non nulle $c$~: $f = \frac1c\Pi$ est un
polynôme. Réciproquement, la question~1 rend tout polynôme
non constant propre. Et $\eu^z$ est honnêtement exclu~: le
long de $\R_-$, $\abs{\eu^z} = \eu^x \to 0$ tandis que
$\abs z \to \infty$.

\textbf{17.} Paramétrer le~\cref{thm:b3:holomorphic:formula}
au \omterm{ex:b3:groups:actions}{centre}~: avec $w = a + r\eu^{\iu\theta}$, $\dd w = \iu
r\eu^{\iu\theta}\dd\theta$,
\[
f(a) = \frac1{2\iu\pi}\int_{C_r}\frac{f(w)}{w - a}\,\dd w
= \frac1{2\pi}\int_0^{2\pi}
f\bigl(a + r\eu^{\iu\theta}\bigr)\,\dd\theta ;
\]
prendre les parties réelles donne la propriété de la moyenne
de $u$. Si $u$ atteint un maximum en un point intérieur du
\omterm{def:b3:topology:connected}{connexe} $\Omega$~: $\abs{\eu^f} = \eu^u$ atteint un maximum
intérieur, donc $\eu^f$ est constante par
le~\cref{thm:b3:holomorphic:maximum}(2), et $u =
\log\abs{\eu^f}$ est constante. Sur un domaine borné avec
continuité jusqu'au bord, $\sup_{\bar\Omega}u =
\sup_{\partial\Omega}u$, exactement comme pour $\abs f$.

\textbf{18.} \emph{Cas $\abs a > r$.} Choisir $R$ avec $r <
R < \abs a$~: sur le disque convexe $D(0, R)$ la fonction $a
- z$ est \omterm{def:b3:holomorphic:holo}{holomorphe} et sans zéro, et $z \mapsto -1/(a - z)$
a une primitive $L$ là
(le~\cref{thm:b3:holomorphic:cauchy})~; après ajustement de
la constante, $\bigl(\eu^{-L}(a - z)\bigr)' =
\eu^{-L}\bigl(-L'\,(a - z) - 1\bigr) = 0$ donne $\eu^L = a -
z$~: un logarithme \omterm{def:b3:holomorphic:holo}{holomorphe} existe, et $\log\abs{a - z} =
\operatorname{Re}L(z)$. Propriété de la moyenne de la
question~17 en $0$, rayon $r$~:
\[
\frac1{2\pi}\int_0^{2\pi}
\log\bigl|a - r\eu^{\iu\theta}\bigr|\,\dd\theta
= \operatorname{Re}L(0) = \log\abs a .
\]
\emph{Cas $\abs a < r$.} De $a - r\eu^{\iu\theta} =
-r\eu^{\iu\theta}\bigl(1 - \frac ar\eu^{-\iu\theta}\bigr)$,
la moyenne égale $\log r$ plus la moyenne de $\log\abs{1 -
\frac ar\eu^{-\iu\theta}}$. La substitution $\theta \mapsto
2\pi - \theta$, puis --- en écrivant $\frac ar =
\rho\eu^{\iu\varphi}$, $\rho < 1$, le cas $a = 0$ étant
trivial --- le décalage $\theta \mapsto \theta - \varphi$
(tous deux préservent les moyennes sur une période)
transforment cela en la moyenne de $\log\abs{1 -
\rho\eu^{\iu\theta}}$~: le premier cas avec $(a, r) = (1,
\rho)$, qui donne $\log 1 = 0$. Total~: $\log r =
\log\max(\abs a, r)$ dans les deux cas.

\textbf{19.} $\log\abs{P(\eu^{\iu\theta})} = \log\abs c +
\sum_i\log\abs{\alpha_i - \eu^{\iu\theta}}$, chaque racine
répétée selon sa multiplicité~; en moyennant en $\theta$ et
en appliquant la question~18 avec $r = 1$ à chaque racine
hors du cercle unité on obtient $\log\bigl(\abs
c\prod_i\max(1, \abs{\alpha_i})\bigr)$. Vérifications. Pour
$P = 2X - 1 = 2(X - \frac12)$ la formule prédit $\log 2$~;
directement, $\abs{2\eu^{\iu\theta} - 1} = 2\abs{\frac12 -
\eu^{\iu\theta}}$ et la moyenne de $\log\abs{\frac12 -
\eu^{\iu\theta}}$ est $\log\max(\frac12, 1) = 0$~: moyenne
$\log 2$. Pour $P = X^2 - X = X(X - 1)$ la racine $1$ est
\emph{sur} le cercle~; la formule prédit $0$. Directement,
la moyenne de $\log\abs{\eu^{\iu\theta}}$ est $0$, et avec
$\abs{\eu^{\iu\theta} - 1} = 2\abs{\sin\frac\theta2}$~:
\[
\frac1{2\pi}\int_0^{2\pi}
\log\Bigl(2\sin\frac\theta2\Bigr)\dd\theta
= \frac1\pi\int_0^\pi\log(2\sin u)\,\dd u
= \log 2 + \frac J\pi,
\qquad J = \int_0^\pi\log\sin u\,\dd u .
\]
La substitution $u = 2v$ et $\sin 2v = 2\sin v\cos v$ donnent
$J = \pi\log2 + 2\int_0^{\pi/2}\log\sin +
2\int_0^{\pi/2}\log\cos = \pi\log 2 + 2J$ (chaque moitié
égale $J/2$ par les symétries de $\sin$), donc $J = -\pi\log
2$ (les intégrales impropres convergent, $\log\sin$ étant
\omterm{def:b3:lebesgue:l1}{intégrable} aux extrémités)~: la moyenne est $\log2 - \log2 =
0$. La formule survit aux racines sur le cercle.

\textbf{20.} $\frac{\eu^z - 1}z =
\sum_{j\geq0}\frac{z^j}{(j+1)!}$ est entière et vaut $1$ en
$0$~: sa réciproque est \omterm{def:b3:holomorphic:holo}{holomorphe} près de $0$ (sur $\abs z
< 2\pi$ en fait, les autres zéros les plus proches de
$\eu^z - 1$ étant $\pm2\iu\pi$), donc $\frac z{\eu^z-1}$ est
analytique en $0$. En multipliant les deux séries et en
lisant le coefficient de $z^n$, $n \geq 1$, dans
$(\frac{\eu^z-1}z)\cdot(\frac z{\eu^z-1}) = 1$~:
\[
\sum_{k=0}^{n}\frac{B_k}{k!\,(n+1-k)!} = 0
\quad\Longleftrightarrow\quad
\sum_{k=0}^{n}\binom{n+1}{k}B_k = 0 .
\]
Successivement~: $B_0 = 1$, $B_1 = -\frac12$, $B_2 =
\frac16$, $B_3 = 0$, $B_4 = -\frac1{30}$, $B_5 = 0$, $B_6 =
\frac1{42}$. Parité~: avec $F(z) = \frac z{\eu^z-1} +
\frac z2$,
\[
F(-z) = \frac{-z}{\eu^{-z} - 1} - \frac z2
= \frac{z\,\eu^z}{\eu^z - 1} - \frac z2
= z + \frac{z}{\eu^z - 1} - \frac z2 = F(z) :
\]
$F$ est paire, donc $B_{2k+1} = 0$ pour $k \geq 1$ (le seul
coefficient impair $B_1$ a été absorbé par $+\frac z2$).
Pointeur vers l'avant~: $\cot w = \iu +
\frac{2\iu}{\eu^{2\iu w}-1}$ donne $w\cot w = 1 +
\sum_{k\geq1}\frac{B_{2k}}{(2k)!}(2\iu w)^{2k}$, donc les
coefficients de Laurent de la cotangente --- d'où, par
le~\cref{ch:b3:residues}, chaque $\zeta(2k)$ --- sont
tarifés par les nombres de Bernoulli~:
\[
\zeta(2k) = (-1)^{k+1}\,\frac{(2\pi)^{2k}B_{2k}}{2\,(2k)!} .
\]

\textbf{21.} Écrire $f = \sum_nc_nz^n$ (rayon $\infty$)~;
alors $g(z) = \overline{f(\bar z)} = \sum_n\bar c_nz^n$ est
entière. Sur $\R$~: $g(x) = \overline{f(x)} = f(x)$, donc
$g$ et $f$ coïncident sur un ensemble à points
d'accumulation dans le \omterm{def:b3:topology:connected}{connexe} $\C$~:
le~\cref{thm:b3:holomorphic:identity} donne $g \equiv f$,
c'est-à-dire $f(\bar z) = \overline{f(z)}$ (de façon
équivalente~: tous les $c_n$ sont réels). Pour un polynôme
réel $P$~: $P(\bar\alpha) = \overline{P(\alpha)} = 0$, et
la même identité appliquée aux dérivées réelles $P', P'',
\dots$ préserve les multiplicités~: les racines non réelles
s'apparient --- l'appariement de la question~8, reprouvé
analytiquement.

\textbf{22.} L'échelle, assemblée~: bornée $\Rightarrow$
constante (11)~; dominée par $A + B\abs z^m$ $\Rightarrow$
polynôme (12)~; partie réelle bornée supérieurement
$\Rightarrow$ constante (13)~; doublement périodique
$\Rightarrow$ constante (14)~; image manquant un disque
$\Rightarrow$ constante (15)~; propre $\Rightarrow$
polynôme (16). Chaque barreau est la formule de Cauchy~: la
valeur en un point est une moyenne de cercle, donc tous les
coefficients de Taylor sont tarifés par la taille de $f$ sur
de grands cercles, et un plafond de croissance anéantit les
coefficients en masse. Rien de tel ne contraint les
fonctions réelles $\mathcal C^\infty$~: une fonction bosselée
(le~\cref{thm:b3:lp:regularization}) est bornée, à support
compact et sauvagement non constante, et ses dérivées en
tout point hors du support s'annulent toutes sans que la
fonction s'annule nulle part à proximité. La régularité ne
couple pas du tout les dérivées en des points distincts~;
l'holomorphie enchaîne chaque dérivée à une seule intégrale
sur un cercle lointain. Une condition locale avec un
informateur global --- voilà pourquoi les fonctions entières
obéissent à la loi et à l'ordre.

\textbf{23.} En développant $\abs{P(\eu^{\iu\theta})}^2 =
\sum_{k,l}a_k\bar a_l \eu^{\iu(k-l)\theta}$ et en moyennant
on tue tout terme $k \neq l$~: la moyenne est
$\sum_k\abs{a_k}^2 =: N$. Supposons d'abord que $P$ n'a pas
de zéro sur le cercle unité, donc $\theta \mapsto
\log\abs{P(\eu^{\iu\theta})}$ est \omterm{def:b3:topology:continuity}{continue}. La borne $\log
t \leq t - 1$ appliquée à $t = \abs P^2/N$ donne, après
moyenne,
\[
\frac1{2\pi}\int_0^{2\pi}\log\abs{P(\eu^{\iu\theta})}^2
\dd\theta - \log N
\;\leq\; \frac1N\cdot N - 1 = 0,
\]
donc la moyenne géométrique de $\abs P$ est au plus $\sqrt
N$~; la question~19 identifie cette moyenne géométrique
comme $\abs c\prod_i\max(1, \abs{\alpha_i})$~: l'inégalité
de Landau. Racines sur le cercle~: choisir $r > 1$ distinct
de tout $\abs{\alpha_i}$~; le polynôme $P(rX)$, de racines
$\alpha_i/r$ hors du cercle unité et de coefficients
$a_kr^k$, satisfait l'inégalité~; les deux membres sont
continus en $r$, et faire $r \to 1^+$ donne le cas général.
Sur $X^2 - X$~: racines $0$ et $1$, donc le membre de gauche
est $1$, et le membre de droite est $\sqrt{1 + 1} =
\sqrt2$~: vrai, avec de la marge.

\textbf{24.} Écrire $\eu^z - 1 = z\,g(z)$ avec $g(z) =
\sum_{k\geq0}\frac{z^k}{(k+1)!}$ entière, $g(0) = 1$. Comme
$\eu^z = 1$ exactement sur $2\pi\iu\Z$, $g$ n'a pas de zéro
dans $D(0, 2\pi)$ (pour $0 < \abs z < 2\pi$ car $\eu^z - 1
\neq 0$, en $0$ par $g(0) = 1$), donc $h = 1/g$ est
\omterm{def:b3:holomorphic:holo}{holomorphe} sur $D(0,2\pi)$ et sa série de Taylor en $0$ ---
par définition $\sum\frac{B_n}{n!}z^n$ --- converge sur tout
le disque~: $\rho \geq 2\pi$. Si $\rho > 2\pi$, la somme $S$
serait \omterm{def:b3:holomorphic:holo}{holomorphe} sur $D(0,\rho)$, et elle coïncide avec $z
\mapsto z/(\eu^z - 1)$ sur $0 < \abs z < 2\pi$~; les deux
sont \omterm{def:b3:holomorphic:holo}{holomorphes} sur l'\omterm{def:b3:topology:topology}{ouvert} \omterm{def:b3:topology:connected}{connexe} $D(0,\rho) \setminus
2\pi\iu\Z$, donc par le théorème d'identité elles
coïncident là. Mais quand $z \to 2\pi\iu$, $\abs{z/(\eu^z -
1)} \to \infty$ (numérateur $\to 2\pi$, dénominateur $\to
0$) tandis que $S$ est \omterm{def:b3:topology:continuity}{continue} en $2\pi\iu$~:
contradiction. D'où $\rho = 2\pi$ exactement, et la formule
de Hadamard donne $\limsup_n\abs{B_n/n!}^{1/n} =
\frac1{2\pi}$~; les coefficients impairs étant nuls à
partir de $n = 3$, le $\limsup$ est porté par les \omterm{def:b3:holomorphic:index}{indices}
pairs, ce qui est la formule énoncée avec $n = 2k$.
Numériquement en $k = 6$~: $(2\pi)^{12} \approx
3{,}7858\cdot10^9$ et $2\cdot12! = 958\,003\,200$, donc
$2\,(2k)!/(2\pi)^{2k} \approx 0{,}25305$, contre
$\abs{B_{12}} = \frac{691}{2730} \approx 0{,}25311$. Le
rapport, $1{,}00025$, est exactement $\zeta(12)$ aux
chiffres affichés~: la formule de calcul des résidus
$\zeta(2k) =
(-1)^{k+1}\frac{(2\pi)^{2k}B_{2k}}{2\,(2k)!}$
du~\cref{ch:b3:residues} explique à la fois le facteur $2$
et le petit excès.

\textbf{25.} Borne de Cauchy~: si $\abs z > 1 + M$ avec $M =
\max_k\abs{q_k}$, alors
\[
\Bigl|\sum_{k<n}q_kz^k\Bigr|
\leq M\,\frac{\abs z^n - 1}{\abs z - 1}
< \frac{M}{\abs z - 1}\,\abs z^n \leq \abs z^n,
\]
donc $\abs{Q(z)} > 0$~: toutes les racines sont dans
$\overline D(0, 1+M)$. Les coefficients des $P_j$
convergent, donc sont bornés par un certain $M$~: toutes les
racines de tous les $P_j$ (et de $P$) sont dans le compact
$K = \overline D(0, 1 + M)$. Soit $v_j \in K^n$ un vecteur
listant les racines de $P_j$ avec multiplicité
(question~7). Toute sous-suite de $(v_j)$ a une
sous-sous-suite convergeant vers un certain $(\beta_1,
\dots, \beta_n)$~; les coefficients de $\prod_i(X -
\alpha_i^{(j)})$ sont, au signe près, les fonctions
symétriques élémentaires de $v_j$ --- \omterm{def:b3:topology:continuity}{continues} --- donc le
long de cette sous-suite ils convergent vers les
coefficients de $\prod_i(X - \beta_i)$~; mais ils
convergent vers ceux de $P$ par hypothèse, donc $\prod_i(X -
\beta_i) = P$~: toute limite sous-séquentielle de $(v_j)$
est une permutation du vecteur de racines de $P$. Si la
distance d'appariement $\delta_j =
\min_\sigma\max_i\,\abs{\alpha_i^{(j)} -
\alpha_{\sigma(i)}}$ ne tendait pas vers $0$, une sous-suite
garderait $\delta_j \geq \varepsilon$ tandis que ses
vecteurs de racines convergent vers une permutation des
racines de $P$ --- forçant $\delta_j \to 0$ le long d'elle~:
contradiction. Donc les multi-ensembles de racines
convergent. Netteté~: $P_\varepsilon = (X - 1)^2 +
\varepsilon$ a pour racines $1 \pm \iu\sqrt\varepsilon$~: la
racine double bouge de $\sqrt\varepsilon$, p.\ ex.\ de
$10^{-2}$ pour $\varepsilon = 10^{-4}$. En général, si
$\alpha$ est une racine $m$-uple, alors près de $\alpha$ on
a $\abs{P(z)} \asymp \abs{z - \alpha}^m$, donc une
perturbation de taille $\varepsilon$ déplace le cluster de
racines d'environ $\varepsilon^{1/m}$~: continuité de Hölder
d'exposant $\frac1m$ et pas mieux --- voilà pourquoi un
solveur numérique près d'une racine double ne retient que la
moitié des chiffres de travail.
\end{solution}
