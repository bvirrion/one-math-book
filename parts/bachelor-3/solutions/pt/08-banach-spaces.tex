\begin{solution}{exo:b3:banach:1}
(um) $\norm{Sx}_2 = \norm x_2$: $ S $ é uma isometria, $\vertiii S =
1$. Para o deslocamento para trás: $\norm{S^*x}_2^2 = \sum_{n \geq
2}\abs{x_n}^2 \leq \norm x_2^2$, com igualdade para $ x = e_2$:
$\vertiii{S^*} = 1$.

(b) $\norm{M_ax}_2^2 = \sum\abs{a_n}^2\abs{x_n}^2 \leq \norm
a_\infty^2\norm x_2^2$; testando $ x = e_n $ dá $\vertiii{M_a}
\geq \abs{a_n}$ para cada $ n $: $\vertiii{M_a} = \norm a_\infty $.

(c) $\abs{\Lambda(f)} \leq \int_0^1\abs f \leq \norm f_\infty $:
$\norm\Lambda \leq 1$. Para $\varepsilon > 0$ deixar $ f_\varepsilon $
ser $1$ sobre $[0, \frac12 - \varepsilon]$, $-1$ sobre $[\frac12 +
\varepsilon, 1]$, afim entre: $\norm{f_\varepsilon}_\infty =
1$ e $\Lambda(f_\varepsilon) \geq 1 - 2\varepsilon $:
$\norm\Lambda = 1$. Não alcançado: $\Lambda(f) = 1$ com $\norm
f_\infty \leq 1$ forças $\int_0^{1/2}f = \frac12$ e
$\int_{1/2}^1 f = -\frac12$, ou seja,\ (continuidade, $\abs f \leq 1$)
$ f \equiv 1$ sobre $[0, \frac12]$ e $ f \equiv -1$ sobre $[\frac12,
1]$: contradição em $\frac12$.
\end{solution}

\begin{solution}{exo:b3:banach:2}
Escrever $ S = T\bigl(I - T^{-1}(T - S)\bigr)$ com
$\vertiii{T^{-1}(T-S)} \leq \vertiii{T^{-1}}\,\vertiii{T - S} =
\theta < 1$: por \cref{prop:b3:banach:neumann}, $ S $ é invertível
com $ S^{-1} = \sum_{n\geq0}\bigl(T^{-1}(T -
S)\bigr)^nT^{-1}$, de onde
\[
\vertiii{S^{-1} - T^{-1}} \leq
\sum_{n\geq1}\theta^n\,\vertiii{T^{-1}}
= \frac{\vertiii{T^{-1}}^2\,\vertiii{T - S}}{1 - \theta}.
\]
Para o sistema linear: $ x_T = T^{-1}b $ e $ x_S = S^{-1}b $
diferem no \omterm{def:b3:rings:primemaximal}{máximo} por esses tempos limitados $\norm b $ --- pequeno
perturbações de um sistema invertível permanecem exclusivamente \omterm{def:b3:groups:derived}{solucionável},
com Lipschitz dependência da solução do operador.
\end{solution}

\begin{solution}{exo:b3:banach:3}
(um) $\ell^1$: deixar $(x^{(k)})$ seja Cauchy. Cada coordenada é
Cauchy ($\abs{x^{(k)}_n - x^{(l)}_n} \leq \norm{x^{(k)} -
x^{(l)}}_1$): deixar $ x_n = \lim_kx^{(k)}_n $. Dado $\varepsilon $,
para $ k, l \geq K $: $\sum_{n \leq N}\abs{x^{(k)}_n - x^{(l)}_n}
\leq \varepsilon $ para cada $ N $; deixar $ l \to \infty $, então $ N \to
\infty $: $\norm{x^{(k)} - x}_1 \leq \varepsilon $, e $ x =
x^{(k)} - (x^{(k)} - x) \in \ell^1$. $\ell^\infty $: Cauchy para
$\norm\cdot_\infty $ é uniformemente Cauchy: converge uniformemente para
uma sequência limitada. $ c_0$ está fechado em $\ell^\infty $: se
$ x^{(k)} \to x $ uniformemente com $ x^{(k)}_n \to_n 0$, então
$\abs{x_n} \leq \norm{x - x^{(k)}}_\infty + \abs{x^{(k)}_n}$
dá $\limsup_n\abs{x_n} \leq \varepsilon $: $ x \in c_0$; um
O subespaço fechado de um espaço de Banach é Banach. Sequências finitas:
seu \omterm{def:b3:topology:interior}{encerramento} contém cada $ x \in c_0$ (truncamentos convergem:
$\sup_{n>N}\abs{x_n} \to 0$) e está contido no fechado
$ c_0$.

(b) Por homogeneidade, assuma $\norm x_p = 1$: então $\abs{x_n} \leq
1$ para todos $ n $, então $\abs{x_n}^q \leq \abs{x_n}^p $ e $\norm
x_q \leq 1 = \norm x_p $; para $ q = \infty $, $\abs{x_n} \leq
\norm x_p $ diretamente. Rigor: $ x_n = n^{-\alpha}$ com
$\frac1q < \alpha \leq \frac1p $ reside em $\ell^q \setminus
\ell^p $ (Série Riemann).
\end{solution}

\begin{solution}{exo:b3:banach:4}
($\leq $) Se $\norm f \leq 1$ e $ f\restriction_F = 0$: para
cada $ y \in F $, $\abs{f(x)} = \abs{f(x - y)} \leq \norm{x -
y}$; pegue o ínfimo. ($\geq $, alcançado)
\cref{cor:b3:banach:hbnormed}(3) produz $ f $ com
$ f\restriction_F = 0$, $\norm f \leq 1$, $ f(x) = d(x, F)$: o
supremo é um \omterm{def:b3:rings:primemaximal}{máximo}. Conseqüência: $ F \subseteq \bigcap\{\ker
f : f\restriction_F = 0\}$ trivialmente, e um ponto $ x \notin F $
é excluído da interseção pelo funcional acima
($ f(x) = d(x,F) > 0$, $ F $ sendo fechado).
\end{solution}

\begin{solution}{exo:b3:banach:5}
Para $ y \in \ell^\infty $: $\abs{\Lambda_y(x)} =
\abs{\sum x_ny_n} \leq \norm y_\infty\norm x_1$, então
$\norm{\Lambda_y} \leq \norm y_\infty $; testando em $ e_n $:
$\abs{y_n} = \abs{\Lambda_y(e_n)} \leq \norm{\Lambda_y}$:
igualdade. Por outro lado, dado $\Lambda \in (\ell^1)'$, definir $ y_n =
\Lambda(e_n)$: $\abs{y_n} \leq \norm\Lambda $, então $ y \in
\ell^\infty $; $\Lambda $ e $\Lambda_y $ concordar com o finito
sequências densas em $\ell^1$ ($\norm{x - \sum_{n\leq
N}x_ne_n}_1 = \sum_{n>N}\abs{x_n} \to 0$): $\Lambda =
\Lambda_y $. O mapa $ y \mapsto \Lambda_y $ é linear, isométrico,
para. Para $(\ell^\infty)'$ o mesmo início produz uma sequência
$ y_n = \Lambda(e_n)$, mas sequências finitas são \emph{não} \omterm{def:b3:topology:interior}{denso}
em $\ell^\infty $
(a sequência constante $\mathbf 1$ está à distância $1$ de todos
deles), então $\Lambda $ não é determinado pelo $ y_n $ --- e
de fato $(\ell^\infty)' \neq \ell^1$
(\cref{pb:b3:banach:1}).
\end{solution}

\begin{solution}{exo:b3:banach:6}
Para cada fixo $ x $, $ y \mapsto B(x, y)$ é \omterm{def:b3:topology:continuity}{contínuo} linear:
$\sup_{\norm y \leq 1}\norm{B(x,y)} < \infty $. Então a família
$\{B(\cdot, y) : \norm y \leq 1\} \subseteq \mathcal L(E, G)$
(cada membro \omterm{def:b3:topology:continuity}{contínuo}, por continuidade em $ x $) é pontual
delimitado pelo espaço de Banach $ E $: Banach--Steinhaus
(\cref{thm:b3:banach:banachsteinhaus}) rendimentos $ C $ com
$\norm{B(x,y)} \leq C\norm x $ para todos $\norm y \leq 1$;
homogeneidade em $ y $ acabamentos: $\norm{B(x,y)} \leq
C\norm x\norm y $.
\end{solution}

\begin{solution}{exo:b3:banach:7}
(um) $\norm f_1 \leq \norm f_\infty $: a identidade é limitada,
e bijetivo. Seu inverso é ilimitado: $ f_n(x) = x^n $ tem
$\norm{f_n}_1 = \frac1{n+1} \to 0$ mas $\norm{f_n}_\infty = 1$.
Nenhuma contradição com \cref{cor:b3:banach:equivnorms}: $(\mathcal
C(\intcc01), \norm\cdot_1)$ é \emph{não} completo
(\cref{exo:b3:complete:1}); o corolário exige completude
em ambos os lados.

(b) No espaço $ E_0$ de sequências finitas (densas em $\ell^2$),
$\varphi(x) = \sum_n n\,x_n $ é linear e ilimitado
($\varphi(e_n) = n $ com $\norm{e_n}_2 = 1$). O gráfico fechado
teorema não se aplica: $ E_0$ não é completo --- e $\varphi $
não tem \omterm{def:b3:topology:continuity}{contínuo} extensão para $\ell^2$, ilustrando que
densidade sem uniforme continuidade é impotente
(\cref{thm:b3:complete:extension}).
\end{solution}

\begin{solution}{exo:b3:banach:8}
Verificamos a hipótese do gráfico fechado. Deixar $ x_k \to x $ e
$ Tx_k \to z $ em $\ell^2$. Para cada $ y $:
\[
\langle z, y\rangle = \lim_k\langle Tx_k, y\rangle
= \lim_k \langle x_k, Ty\rangle = \langle x, Ty\rangle
= \langle Tx, y\rangle,
\]
usando continuidade do produto interno em cada slot
(Cauchy--Schwarz) e simetria duas vezes. Então $ z - Tx $ é ortogonal
para cada $ y $, em particular para si mesmo: $ z = Tx $. O gráfico é
fechado e $\ell^2$ é Banach: $ T $ é limitado
(\cref{thm:b3:banach:closedgraph}). Portanto, um operador simétrico
definido em \emph{todos} de $\ell^2$ é automaticamente limitado;
operadores simétricos genuinamente ilimitados (posição, momento,
Hamiltonianos) devem viver em subespaços \omterm{def:b3:topology:interior}{densos} adequados.
\end{solution}

\begin{solution}{exo:b3:banach:9}
Primeiro, $\norm{\Lambda_n} = \sum_i\abs{w_{i,n}}$: $\leq $ é o
desigualdade triangular; $\geq $ testando um linear por partes $ f $
com $\norm f_\infty \leq 1$ e $ f(x_{i,n}) =
\operatorname{sign}(w_{i,n})$ (interpolar linearmente entre os
um número finito de nós; onde os nós coincidem, os sinais concordam).

($\Rightarrow $) Convergência pontual em cada $ f $ implica (eu),
e limitação pontual, então Banach-Steinhaus
(\cref{thm:b3:banach:banachsteinhaus}) no Banach $\mathcal
C(\intcc01)$ dá (ii).

($\Leftarrow $) Deixar $ M = \sup_n\norm{\Lambda_n} + 1$. Dado $ f $
e $\varepsilon $, escolha um polinômio $ P $ com $\norm{f -
P}_\infty < \varepsilon/(2M)$
(\cref{cor:b3:complete:weierstrass}); então
\[
\Bigl|\Lambda_n f - \int_0^1 f\Bigr|
\leq \abs{\Lambda_n(f - P)} + \Bigl|\Lambda_nP -
\int_0^1P\Bigr| + \Bigl|\int_0^1(P - f)\Bigr|
\leq \varepsilon + \Bigl|\Lambda_nP - \int_0^1P\Bigr| \to
\varepsilon .
\]
Pesos positivos, exatidão em constantes: $\sum_i\abs{w_{i,n}}
= \sum_iw_{i,n} = \Lambda_n(\mathbf 1) = \int_0^1 1 = 1$ para
regras exatas em constantes --- (ii) valem para constantes $1$.
\end{solution}

\begin{solution}{exo:b3:banach:10}
Linearidade de $ J $ é formal; $\norm{J(x)} = \sup_{\norm f \leq
1}\abs{f(x)} = \norm x $ por
\cref{cor:b3:banach:hbnormed}(2). Se $\dim E = n $: $\dim E' =
n $ (uma \omterm{def:b3:topology:basis}{base} fornece funcionais de coordenadas), então $\dim E'' = n $,
e o injetivo (isométrico) $ J $ está em. Separabilidade: deixe
$(f_n)$ ser \omterm{def:b3:topology:interior}{denso} em $ E'$ e escolha $\norm{x_n} = 1$ com
$\abs{f_n(x_n)} \geq \frac12\norm{f_n}$. Deixar $ F =
\overline{\operatorname{Vect}}(x_n)$; se $ F \neq E $, pegar $ g
\in E'$, $ g \neq 0$, desaparecendo em $ F $
(\cref{cor:b3:banach:hbnormed}(3)); escolher $ f_{n_k} \to g $:
\[
\norm{f_{n_k} - g} \geq \abs{(f_{n_k} - g)(x_{n_k})}
= \abs{f_{n_k}(x_{n_k})} \geq \tfrac12\norm{f_{n_k}}
\geq \tfrac12\bigl(\norm g - \norm{f_{n_k} - g}\bigr),
\]
então $\norm{f_{n_k} - g} \geq \frac13\norm g > 0$: contradição.
Por isso $ F = E $ e racional (ou $\Q + \iu\Q $) combinações de
o $ x_n $ formar um contável conjunto \omterm{def:b3:topology:interior}{denso}.
\end{solution}

\begin{solution}{exo:b3:banach:12}
(um) $ W = \ker P $ está fechado (pré-imagem de $0$ sob um
mapa \omterm{def:b3:topology:continuity}{contínuo}); $ V = \operatorname{im}P = \ker(I - P)$
(de fato $ Px = x $ se $ x \in \operatorname{im}P $, usando $ P^2 =
P $), fechado da mesma forma. Todo $ x $ divide como $ Px + (x - Px)$
com $ Px \in V $, $ x - Px \in W $, e $ V \cap W = 0$ ($ x = Px
= 0$): $ E = V \oplus W $.

(b) O argumento do gráfico: seja $ x_n \to x $ e $ Px_n \to z $.
Então $ z \in V $ ($ V $ fechado, $ Px_n \in V $) e $ x_n - Px_n \to
x - z \in W $ ($ W $ fechado). Então $ x = z + (x - z)$ com $ z \in
V $, $ x - z \in W $; pela singularidade da decomposição, $ z =
Px $. O gráfico de $ P $ está fechado, $ E $ é Banach: $ P $ é
limitado (\cref{thm:b3:banach:closedgraph}).

(c) (a) e (b) juntos são a equivalência. Em um Hilbert
espaço todo fechado $ V $ tem o complemento fechado $ V^\perp $
(\cref{ch:b3:hilbert}): todo subespaço fechado é
complementado. Em geral, espaços Banach isso falha --- $ c_0$
não tem complemento fechado em $\ell^\infty $ (Phillips'
teorema, além de nossas ferramentas) --- então projeções limitadas são um
privilégio, e o teorema do gráfico fechado é exatamente o
contabilidade que converte divisões geométricas em delimitadas
operadores.
\end{solution}

\begin{solution}{pb:b3:banach:1}
\textbf{1.} Para $ a, b > 0$: por concavidade de $\log $,
$\log\bigl(\tfrac{a^p}p + \tfrac{b^q}q\bigr) \geq \tfrac1p\log
a^p + \tfrac1q\log b^q = \log(ab)$; exponenciar. (Se $ ab = 0$
a desigualdade é trivial.) Igualdade se $ a^p = b^q $.

\textbf{2.} Podemos assumir $\norm x_p = \norm y_q = 1$
(homogeneidade; zero casos triviais). Então
\[
\abs{\langle x, y\rangle} \leq \sum_n\abs{x_n}\abs{y_n}
\leq \sum_n\Bigl(\frac{\abs{x_n}^p}p +
\frac{\abs{y_n}^q}q\Bigr) = \frac1p + \frac1q = 1 =
\norm x_p\norm y_q .
\]
A igualdade exige $\abs{x_n}^p = \abs{y_n}^q $ para todos $ n $
(caso de igualdade de Young) e alinhamento das fases de
$ x_ny_n $.

\textbf{3.} $\abs{x_n + y_n}^p \leq \bigl(\abs{x_n} +
\abs{y_n}\bigr)\abs{x_n + y_n}^{p-1}$; somando e aplicando
Hölder ($ p $ contra $ q $, observando $(p - 1)q = p $):
\[
\norm{x + y}_p^p \leq \bigl(\norm x_p + \norm
y_p\bigr)\,\Bigl(\sum_n\abs{x_n + y_n}^{p}\Bigr)^{1/q}
= \bigl(\norm x_p + \norm y_p\bigr)\,\norm{x+y}_p^{p/q};
\]
se $\norm{x + y}_p \neq 0$ (outro trivial), divida por
$\norm{x+y}_p^{p/q}$ e usar $ p - \frac pq = 1$. (Finitude de
$\norm{x+y}_p $ primeiro: $\abs{x_n+y_n}^p \leq
2^p(\abs{x_n}^p + \abs{y_n}^p)$.)

\textbf{4.} Completude: quanto a $\ell^1$
(\cref{exo:b3:banach:3}), limites coordenados mais o
cauda uniforme vinculada $\sum_{n\leq N}\abs{x^{(k)}_n - x^{(l)}_n}^p
\leq \varepsilon^p $, deixando $ l $ então $ N $ tendem ao infinito.
Densidade de sequências finitas: $\norm{x - \sum_{n\leq
N}x_ne_n}_p^p = \sum_{n > N}\abs{x_n}^p \to 0$.

\textbf{5.} Hdá mais velho $\abs{\Lambda_y(x)} \leq \norm
x_p\norm y_q $: $\norm{\Lambda_y} \leq \norm y_q $. Teste: deixe
$ x^{(N)}_n = \abs{y_n}^{q-1}\overline{\operatorname{sign}}(y_n)$
para $ n \leq N $, $0$ além (com $\operatorname{sign}$ o
fase unimodular, de modo que $ x_ny_n = \abs{y_n}^q $). Então
$\Lambda_y(x^{(N)}) = \sum_{n\leq N}\abs{y_n}^q $ e
$\norm{x^{(N)}}_p = \bigl(\sum_{n\leq
N}\abs{y_n}^{q}\bigr)^{1/p}$ (como $(q-1)p = q $), então
\[
\norm{\Lambda_y} \geq \Bigl(\sum_{n\leq
N}\abs{y_n}^q\Bigr)^{1 - 1/p} \xrightarrow[N\to\infty]{}
\norm y_q .
\]

\textbf{6.} Definir $ y_n = \Lambda(e_n)$. Com o mesmo teste
vetores, $\sum_{n \leq N}\abs{y_n}^q = \Lambda(x^{(N)}) \leq
\norm\Lambda\,\bigl(\sum_{n\leq N}\abs{y_n}^q\bigr)^{1/p}$,
de onde $\bigl(\sum_{n\leq N}\abs{y_n}^q\bigr)^{1/q} \leq
\norm\Lambda $ para cada $ N $: $ y \in \ell^q $, $\norm y_q \leq
\norm\Lambda $. Os funcionais $\Lambda $ e $\Lambda_y $ concordo
nas sequências finitas densas (questão 4): $\Lambda =
\Lambda_y $. Com a questão 5, $ y \mapsto \Lambda_y $ é um
isomorfismo isométrico $\ell^q \cong (\ell^p)'$.

\textbf{7.} Deixar $\xi \in (\ell^p)''$. Compondo com o
isometria $\ell^q \cong (\ell^p)'$ da questão 6, $\xi $ define
um elemento de $(\ell^q)'$, que (questão 6 com $ p, q $
trocado) é $\Lambda_z $ para um único $ z \in \ell^p $: para cada
$ y \in \ell^q $, $\xi(\Lambda_y) = \sum_nz_ny_n $. Por outro lado $ J(z)(\Lambda_y) = \Lambda_y(z) = \sum_ny_nz_n $: o mesmo
valor. Uma vez que cada elemento de $(\ell^p)'$ é algum $\Lambda_y $,
$\xi = J(z)$: $ J $ está em --- $\ell^p $ é \omterm{rem:b3:banach:bidual}{reflexivo}.

\textbf{8.} Sequências finitas com racionais (reais e imaginárias)
as entradas são contáveis e densas em $\ell^p $ ($ p < \infty $) e
em $ c_0$. Em $\ell^\infty $: a família $\{\mathbf 1_A : A
\subseteq \N\}$ é incontável com $\norm{\mathbf 1_A -
\mathbf 1_B}_\infty = 1$ para $ A \neq B $; as bolas $ B(\mathbf
1_A, \frac12)$ são disjuntos aos pares e um conjunto \omterm{def:b3:topology:interior}{denso} deve se encontrar
cada: não contável conjunto \omterm{def:b3:topology:interior}{denso} existe.

\textbf{9.} Se $\ell^1$ eram \omterm{rem:b3:banach:bidual}{reflexivo}, então $(\ell^\infty)'
\cong ((\ell^1)')' = J(\ell^1)$ seria \omterm{def:b3:galois:separable}{separável} (isométrico
imagem do \omterm{def:b3:galois:separable}{separável} $\ell^1$); por
\cref{exo:b3:banach:10}, a separabilidade do \omterm{def:b3:banach:operator}{dual}
$(\ell^\infty)'$ forçaria $\ell^\infty $ \omterm{def:b3:galois:separable}{separável} ---
contradizendo a questão 8. Então $\ell^1$ não é \omterm{rem:b3:banach:bidual}{reflexivo} (e
$(\ell^\infty)'$ é estritamente maior que $\ell^1$, como pergunta
11 torna concreto).

\textbf{10.} Homogeneidade de $ p $ é claro; subaditividade:
as médias são lineares e $\limsup(u_n + v_n) \leq \limsup u_n +
\limsup v_n $. Sobre $ c $: o Cesàro significa um convergente
sequência converge para o seu limite, então $ p(x) = \lim x =
\mathrm{LIM}(x)$ lá; em particular $\mathrm{LIM} \leq p $ sobre
$ c $. Hahn-Banach (\cref{thm:b3:banach:hahnbanach}) estende
$\mathrm{LIM}$ para $\ell^\infty_\R $ com $\mathrm{LIM} \leq p $
globalmente. Positividade: para $ x \geq 0$, $-\mathrm{LIM}(x) =
\mathrm{LIM}(-x) \leq p(-x) = \limsup\text{avg}(-x) \leq 0$.
Invariância de deslocamento: as médias de $ x - Sx $ telescópio para
$\frac{x_1 - x_{n+1}}n \to 0$, então $ p(\pm(x - Sx)) = 0$ e
$\mathrm{LIM}(x - Sx) = 0$. Limites: $\mathrm{LIM}(x) \leq p(x)
\leq \limsup x $ (as médias ficam atrás do sup), e aplicando isso
para $-x $ fornece o limite inferior.

\textbf{11.} $ e_n \in c_0$, então $\mathrm{LIM}(e_n) = 0$ para todos
$ n $. Se $\mathrm{LIM} = \Lambda_y $ com $ y \in \ell^1$, então
$ y_n = \Lambda_y(e_n) = 0$ para todos $ n $: $\Lambda_y = 0$; mas
$\mathrm{LIM}(\mathbf 1) = 1$. Então $\mathrm{LIM} \in
(\ell^\infty)' \setminus \{\Lambda_y : y \in \ell^1\}$:
$(\ell^\infty)' \neq \ell^1$, de novo.

\textbf{12.} Para $ x = (0,1,0,1,\dots)$: $ x + Sx = \mathbf 1$,
então $2\,\mathrm{LIM}(x) = \mathrm{LIM}(x) + \mathrm{LIM}(Sx) =
1$: $\mathrm{LIM}(x) = \frac12$. Se $\varphi $ eram um
extensão multiplicativa invariante ao deslocamento do limite: $ x\cdot
Sx = 0$ dá $\varphi(x)\varphi(Sx) = \varphi(x)^2 = 0$, então
$\varphi(x) = 0$; mas $\varphi(x) + \varphi(Sx) =
\varphi(\mathbf 1) = 1$ dá $2\varphi(x) = 1$: contradição.
A média e a multiplicação não podem coexistir.

\textbf{13.} Deixar $\norm{x^{(k)}}_p \leq M $. As coordenadas são
limitado por $ M $: uma extração diagonal dá uma subsequência
(ainda escrito $ x^{(k)}$) com $ x^{(k)}_n \to x_n $ para cada
$ n $. Então $ x \in \ell^p $: $\sum_{n\leq N}\abs{x_n}^p =
\lim_k\sum_{n\leq N}\abs{x^{(k)}_n}^p \leq M^p $ para todos $ N $.
Convergência fraca: para $ y \in \ell^q $ e $ N $ arbitrário,
\[
\abs{\Lambda_y(x^{(k)} - x)} \leq
\Bigl|\sum_{n \leq N}(x^{(k)}_n - x_n)y_n\Bigr|
+ 2M\Bigl(\sum_{n>N}\abs{y_n}^q\Bigr)^{1/q},
\]
onde o primeiro termo tende a $0$ como $ k \to \infty $ (finitamente
muitas coordenadas) e a segunda é pequena para $ N $ grande
(Hölder na cauda): $\Lambda_y(x^{(k)}) \to \Lambda_y(x)$
para cada $ y $ --- convergência fraca, já que todo funcional é um
$\Lambda_y $ (questão 6). Em $\ell^1$ isso falha: considere
$(e_n)$, limitado. Qualquer subsequência $(e_{n_k})$ converge
coordenadas para $0$, então seu único candidato a limite fraco é $0$;
mas testando contra $ y \in \ell^\infty $ definido por $ y_{n_k} =
(-1)^k $ (e $0$ em outro lugar), $\Lambda_y(e_{n_k}) = (-1)^k $
diverge. Nenhuma subsequência fracamente convergente: fraca sequencial
compacidade de bolas caracteriza o \omterm{rem:b3:banach:bidual}{reflexivo} mundo.

\textbf{14.} Para $ y \in \ell^1$, $\Lambda_y(x) = \sum x_ny_n $
é definido em $ c_0$ com $\abs{\Lambda_y(x)} \leq \norm
x_\infty\norm y_1$ e testando em $ x^{(N)} =
(\operatorname{sign}y_1, \dots, \operatorname{sign}y_N, 0,
\dots) \in c_0$ dá $\Lambda_y(x^{(N)}) =
\sum_{n\leq N}\abs{y_n} \to \norm y_1$:
$\norm{\Lambda_y} = \norm y_1$. Por outro lado, para $\Lambda \in
(c_0)'$ colocar $ y_n = \Lambda(e_n)$; os mesmos testes dão
$\sum_{n \leq N}\abs{y_n} = \Lambda(x^{(N)}) \leq
\norm\Lambda $, então $ y \in \ell^1$, e $\Lambda = \Lambda_y $ ligado
as sequências finitas densas, portanto, em todos os lugares. A cadeia de
duplas: $(c_0)' = \ell^1$, $(\ell^1)' = \ell^\infty $
(\cref{exo:b3:banach:10}), $(\ell^\infty)' \supsetneq \ell^1$
(questões 9-11): a reflexividade falha logo no primeiro passo
--- $ c_0'' = \ell^\infty \neq c_0$ --- e nunca se recupera.

\textbf{15.} $ J x^{(k)} \in E'' = (E')'$ é uma família de
funcionais limitados no \emph{Banach} espaço $ E'$ (duplas
são completo, \cref{prop:b3:banach:llcomplete}); para cada
$\Lambda \in E'$, a sequência $ Jx^{(k)}(\Lambda) =
\Lambda(x^{(k)})$ converge, portanto é limitado.
Banach-Steinhaus em $ E'$ dá $\sup_k\norm{Jx^{(k)}} <
\infty $, e $ J $ é isométrico
(\cref{rem:b3:banach:bidual}): $\sup_k\norm{x^{(k)}} <
\infty $.

\textbf{16.} ($\Rightarrow $) Limitação é a questão 15;
coordenadas são os funcionais $\Lambda_{e_n}$.
($\Leftarrow $) Deixar $ M = \sup_k\norm{x^{(k)}}_p $, $ y \in
\ell^q $, $\varepsilon > 0$; escolher $ N $ com
$\bigl(\sum_{n>N}\abs{y_n}^q\bigr)^{1/q} < \varepsilon $. Então
\[
\abs{\Lambda_y(x^{(k)} - x)} \leq
\sum_{n\leq N}\abs{x^{(k)}_n - x_n}\,\abs{y_n} +
\norm{x^{(k)} - x}_p\,\varepsilon
\leq \sum_{n\leq N}\abs{x^{(k)}_n - x_n}\abs{y_n} +
(M + \norm x_p)\,\varepsilon,
\]
e a soma finita tende a $0$: $\limsup \leq (M + \norm
x_p)\varepsilon $ para cada $\varepsilon $. (Que $ x \in
\ell^p $ com $\norm x_p \leq M $ segue o estilo Fatou
limites de seção finita: $\sum_{n \leq N}\abs{x_n}^p =
\lim_k\sum_{n\leq N}\abs{x^{(k)}_n}^p \leq M^p $.) Para $ e_k $
em $\ell^2$: limitado, coordenado $\to 0$, então $ e_k
\rightharpoonup 0$, ainda $\norm{e_k} = 1$: a unidade de massa
escapa para um índice infinito, invisível para todos os fixos
funcional.

\textbf{17.} Pegar $\Lambda $ com $\norm\Lambda = 1$ e
$\Lambda(x) = \norm x $ (\cref{cor:b3:banach:hbnormed}). Então
$\norm x = \lim\Lambda(x^{(k)}) \leq
\liminf\norm\Lambda\,\norm{x^{(k)}} =
\liminf\norm{x^{(k)}}$. (Com $ e_k \rightharpoonup 0$: $0
\leq \liminf 1$, e a desigualdade pode ser estrita.)

\textbf{18.} Em $\ell^2$, $\norm{x^{(k)} - x}_2^2 =
\norm{x^{(k)}}_2^2 - 2\operatorname{Re}\langle x^{(k)},
x\rangle + \norm x_2^2$ (caso real: $-2\langle x^{(k)},
x\rangle $). Convergência fraca aplicada ao funcional
$\Lambda_x $ dá $\langle x^{(k)}, x\rangle \to \norm
x_2^2$, e as normas convergem por hipótese: o lado direito
tende a $\norm x^2 - 2\norm x^2 + \norm x^2 = 0$.
Contra-exemplo sem convergência de norma: $ e_k
\rightharpoonup 0$, $\norm{e_k - 0} = 1 \not\to 0$.

\textbf{19.} Convergência de coordenadas: aplique os funcionais
$\Lambda_{e_n} \in (\ell^1)' = \ell^\infty $ ($ y = e_n $).
Construção: tendo escolhido $ k_{j-1}, N_{j-1}$, escolha $ k_j >
k_{j-1}$ tão grande que $\sum_{n \leq
N_{j-1}}\abs{x^{(k_j)}_n} < \frac\delta{20}$ (finitamente muitos
coordenadas, cada $\to 0$), então $ N_j > N_{j-1}$ tão grande
que a cauda satisfaz $\sum_{n > N_j}\abs{x^{(k_j)}_n} <
\frac\delta{20}$ (convergência da série que define
$\norm{x^{(k_j)}}_1$). O bloco $ B_j = \intoc{N_{j-1}}{N_j}$
então carrega tudo menos $\frac\delta{10}$ da massa de
$ x^{(k_j)}$.

\textbf{20.} Com $ y $ conforme definido ($\abs{y_n} \leq 1$
em todos os lugares):
\[
\langle x^{(k_j)}, y\rangle
= \sum_{n \in B_j}\abs{x^{(k_j)}_n}
+ \sum_{n \notin B_j}x^{(k_j)}_ny_n
\geq \Bigl(\norm{x^{(k_j)}}_1 - \frac\delta{10}\Bigr) -
\frac\delta{10}
\geq \delta - \frac{2\delta}{10} = \frac{4\delta}5 > 0,
\]
a desigualdade média porque a massa fora do bloco está em
mais $\frac\delta{10}$ (questão 19). Mas $ y \in
\ell^\infty = (\ell^1)'$ e $ x^{(k)} \rightharpoonup 0$
vigor $\langle x^{(k_j)}, y\rangle \to 0$: contradição.
Portanto, sequências fracamente nulas de $\ell^1$ são norma-nulos, e por
tradução fracamente convergentes convergem em norma: Schur's
teorema.

\textbf{21.} Uma subsequência fracamente convergente de $(e_k)$ faria
tem limite $0$ (coordenadas), portanto por Schur $\norm{e_{k_j}}_1
\to 0$ --- mas as normas são $1$. Portanto, não há fracamente convergente
existe subsequência, conforme encontrado à mão na questão 13. Não
paradoxo: Schur diz que as sequências não conseguem distinguir o fraco
da norma \omterm{def:b3:topology:topology}{topologia} em $\ell^1$ (as próprias \omterm{def:b3:topology:topology}{topologias}
diferem --- fraco bairros nunca são limitados por normas) e
fraco \omterm{thm:b3:topology:metriccompact}{compacidade sequencial} da bola unitária é diferente,
propriedade mais forte, equivalente à reflexividade
(Eberlein--\v Smulian, além de nossas ferramentas; o fracasso, em
pelo menos, nós provamos).

\textbf{22.} \emph{O censo.}
\begin{center}
\begin{tabular}{l|l|c|c|c}
$ E $ & $ E'$ & sep. & refl. & weak seq.\ cpt.\ balls\\
\hline
$ c_0$ & $\ell^1$ & yes & no & no ($ e_1{+}\dots{+}e_k $)\\
$\ell^1$ & $\ell^\infty $ & yes & no & no ($ e_k $, q.~21)\\
$\ell^p $ & $\ell^q $ & yes & yes & yes (q.~13)\\
$\ell^\infty $ & $\supsetneq\ell^1$ & no & no & no\\
\end{tabular}
\end{center}
Assinaturas: $ c_0$ --- seu bidual é $\ell^\infty $: o primeiro
não \omterm{rem:b3:banach:bidual}{reflexivo} etapa (questão 14); $\ell^1$ --- Schur's
propriedade (questão 20); $\ell^p $ --- reflexividade e fraco
compacidade (questões 7, 13); $\ell^\infty $ ---
não separabilidade e limites de Banach: funcionais sem sequência
pode representar (questões 8, 10--11). Uma família de espaços,
quatro mundos diferentes.

\textbf{23.} Deixar $(f_k) \subseteq F $ com $\norm{x - f_k}
\to d $. Então $\norm{f_k} \leq \norm x + \sup_k\norm{x -
f_k}$: limitado, então pela questão 13 uma subsequência $ f_{k_j}
\rightharpoonup f $. Se $ f \notin F $, então $\delta =
\operatorname{dist}(f, F) > 0$ ($ F $ fechado); sobre $ F \oplus \R
f $ a forma linear $\lambda(g + tf) = t $ satisfaz
$\abs{\lambda(u)} \leq \norm u/\delta $ (porque $\norm{g +
tf} \geq \abs t\,\delta $), e Hahn-Banach estende-o para
$\Lambda \in (\ell^p)'$ com $\Lambda\restriction_F = 0$,
$\Lambda(f) = 1$; mas então $0 = \Lambda(f_{k_j}) \to
\Lambda(f) = 1$: contradição. Então $ f \in F $, e $ x -
f_{k_j} \rightharpoonup x - f $ dá, pela questão 17,
\[
d \leq \norm{x - f} \leq \liminf_j\,\norm{x - f_{k_j}} = d :
\]
a distância é alcançada em $ f $. Em $ c_0$: $\abs{\Lambda(x)}
\leq \sum_n2^{-n}\abs{x_n} < \norm x_\infty $ para cada $ x
\neq 0$ (uma sequência nula diferente de zero não pode satisfazer $\abs{x_n} =
\norm x_\infty $ para todos $ n $), enquanto os truncados $(1,
\dots, 1, 0, \dots)$ dar $\Lambda = 1 - 2^{-N} \to 1$: então
$\norm\Lambda = 1$, nunca alcançado. Fórmula de distância: para $ f
\in \ker\Lambda $, $\abs{\Lambda(x)} = \abs{\Lambda(x - f)}
\leq \norm{x - f}$, então $\operatorname{dist}(x, \ker\Lambda)
\geq \abs{\Lambda(x)}$; inversamente, para $ u $ na bola unitária
com $\Lambda(u) \geq 1 - \varepsilon $, o vetor $ f = x -
\frac{\Lambda(x)}{\Lambda(u)}u $ reside em $\ker\Lambda $ com
$\norm{x - f} \leq \frac{\abs{\Lambda(x)}}{1 - \varepsilon}$:
igualdade. Se algum $ f \in \ker\Lambda $ alcançou isso, $ z = x -
f $ satisfaria $\abs{\Lambda(z)} = \abs{\Lambda(x)} =
\norm z \neq 0$, então $\Lambda $ atingiria sua norma em
$ z/\norm z $: impossível. Um hiperplano fechado de $ c_0$ sem
pontos mais próximos em qualquer lugar --- a reflexividade não era decorativa.

\textbf{24.} Definir $ y_1 = x^{(1)}$. Dado $ y_1, \dots, y_j $,
cada mapa $ k \mapsto \langle y_i, x^{(k)}\rangle $ tende a $0$
($ y_i \in \ell^2 = (\ell^2)'$), então há $ k_{j+1}$ além
o índice anterior com $\abs{\langle y_i,
x^{(k_{j+1})}\rangle} \leq \frac1{j+1}$ para $ i = 1, \dots,
j $; ligue para a escolha $ y_{j+1}$. Então
\[
\Bigl\lVert\sum_{j=1}^my_j\Bigr\rVert_2^2
= \sum_{j=1}^m\norm{y_j}_2^2
+ 2\sum_{j=2}^m\sum_{i<j}\langle y_i, y_j\rangle
\leq mC^2 + 2\sum_{j=2}^m\frac{j-1}j
\leq mC^2 + 2m,
\]
e dividindo por $ m^2$: $\norm{\frac1m\sum_jy_j}_2^2 \leq
\frac{C^2 + 2}m \to 0$. (Para um limite fraco $ x \neq 0$, aplicar
isso para $ x^{(k)} - x $.) No ortonormal $(e_k)$ não
a extração é ainda necessária: $\norm{\frac1m(e_1 + \dots +
e_m)}_2 = \frac{\sqrt m}m = \frac1{\sqrt m}$. Médias
converter convergência fraca em convergência de norma: o
Banach--Saks propriedade de $\ell^2$.

\textbf{25.} $ A_mx = \frac1m\sum_{i=0}^{m-1}S^ix $, então
linearidade e invariância de deslocamento fornecem $ L(A_mx) = L(x)$. Para qualquer
limitado $ u $ e $\varepsilon > 0$, escolha $ N $ com $ u_n \leq
\limsup u + \varepsilon $ para $ n \geq N $; positividade aplicada
para $(\limsup u + \varepsilon)\mathbf 1 - S^Nu \geq 0$ e
$ L(S^Nu) = L(u)$ dar $ L(u) \leq \limsup u + \varepsilon $,
e simetricamente $ L(u) \geq \liminf u - \varepsilon $: por isso
$\liminf A_mx \leq L(x) \leq \limsup A_mx $ para cada $ m $. Se
$ x $ é $ T $-periódico, $ A_Tx $ é a sequência constante igual a
o período significa $\mu $: $ L(x) = \mu $ para \emph{todo} Banach
limite --- $\frac13$ sobre $(1,0,0,\dots)$, $\frac12$ sobre
$(0,1,0,1,\dots)$ como na questão 12. Para a sequência de blocos:
em $ N = 3^j $ com $ j $ mesmo o último bloco é todo um, então o
Cesàro significa é $\geq \frac{3^j - 3^{j-1}}{3^j} = \frac23$;
no $ N = 3^j $ com $ j $ estranho todos aqueles que se sentam $\intoc0
{3^{j-1}}$, então a média é $\leq \frac13$. Por isso $ p(x) \geq
\frac23$ e $-p(-x) = \liminf_n\frac{x_1 + \dots + x_n}n
\leq \frac13$. Sobre $ M = c \oplus \R x $ definir $\Lambda_+(y +
tx) = \lim y + t\,p(x)$. Dominação por $ p $: para $ t > 0$,
sublinearidade dá $ p(tx) \leq p(y + tx) + p(-y)$, ou seja\
$ p(y + tx) \geq t\,p(x) + \lim y $ (observação $ p(\pm y) = \pm\lim
y $ para $ y \in c $: Cesàro meio de uma sequência convergente
convergem para o seu limite); para $ t = -s < 0$, $ p(y) \leq p(y -
sx) + p(sx)$ dá $ p(y - sx) \geq \lim y - s\,p(x)$; para $ t
= 0$ existe igualdade. Então $\Lambda_+ \leq p $ sobre $ M $ e
Hahn--Banach estende-o para $ L_+ \leq p $ sobre $\ell^\infty_\R $,
que é um limite de Banach exatamente como na questão 10 (dominação
por $ p $ produz positividade, invariância de deslocamento e o valor
$\lim $ sobre $ c $), com $ L_+(x) = p(x) \geq \frac23$. O mesmo
cálculo com $\Lambda_-(y + tx) = \lim y - t\,p(-x)$
(usando $ p(-sx) \leq p(y - sx) + p(-y)$ para o caso $ t = -s
< 0$) produz um limite de Banach $ L_-$ com $ L_-(x) = -p(-x) \leq
\frac13$. Dois limites de Banach, uma sequência, dois valores:
fora do periódico (e, mais geralmente, \emph{quase
convergente}) mundo, um limite de Banach é uma escolha genuína.
\end{solution}
