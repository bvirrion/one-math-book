\begin{solution}{exo:g10:reffunc:2}
Door $(2,1)$ en $(5,10)$: $m = \frac{10 - 1}{5 - 2} = 3$; dan
$1 = 3 \times 2 + p$ geeft $p = -5$: $f(x) = 3x - 5$.

Door $(-1,4)$ en $(3,-4)$: $m = \frac{-4 - 4}{3 - (-1)} =
\frac{-8}{4} = -2$; dan $4 = -2 \times (-1) + p$ geeft $p = 2$:
$f(x) = -2x + 2$.
\end{solution}

\begin{solution}{exo:g10:reffunc:3}
$(3.1)^2 < (3.2)^2$: het kwadraat stijgt op positieven.

$(-2.7)^2 < (-2.8)^2$: het kwadraat daalt op negatieven en
$-2.8 < -2.7$.

$\frac{1}{5.1} > \frac{1}{5.2}$: de omgekeerde daalt op positieven.

$\sqrt{17} > \sqrt{15}$: de vierkantswortel stijgt.
\end{solution}

\begin{solution}{exo:g10:reffunc:4}
$x^2 = 16$: twee oplossingen, $x = 4$ en $x = -4$ (de horizontale lijn
$y = 16$ snijdt de parabool twee keer).

$x^2 \leq 16$: de parabool ligt onder de lijn tussen de twee snijpunten:
$x \in \intcc{-4}{4}$.

$x^2 > 9$: de parabool ligt strikt boven $y = 9$ buiten
$\intcc{-3}{3}$: $x \in \intoo{-\infty}{-3} \cup \intoo{3}{+\infty}$.
\end{solution}

\begin{solution}{exo:g10:reffunc:5}
Voor $0 < x < 1$: herhaald vermenigvuldigen van $x < 1$ met $x$ geeft
$x^3 < x^2 < x$, en \cref{prop:g10:reffunc:compare} geeft $x < \sqrt x$,
dus
\[
x^3 < x^2 < x < \sqrt x .
\]
Controle met $x = 0.25$: $x^3 = 0.015625$, $x^2 = 0.0625$, $x = 0.25$,
$\sqrt x = 0.5$.
\end{solution}

\begin{solution}{exo:g10:reffunc:6}
$\frac1x = 2$ heeft de unieke oplossing $x = \frac12$ (de hyperbool snijdt
de horizontale lijn $y = 2$ één keer, op de positieve tak).

Voor $x > 0$: $\frac1x < 2$. Vermenigvuldigen met $x > 0$: $1 < 2x$, dus
$x > \frac12$: oplossingsverzameling $\intoo{\frac12}{+\infty}$.

Als $x < 0$ is toegestaan: elke negatieve $x$ voldoet aan
$\frac1x < 0 < 2$, dus de volledige oplossingsverzameling is
$\intoo{-\infty}{0} \cup \intoo{\frac12}{+\infty}$.
\end{solution}

\begin{solution}{exo:g10:reffunc:7}
(a) Op $\intcc{2}{5}$ zijn alle inputs positief en stijgt kwadrateren:
$4 \leq x^2 \leq 25$.

(b) Op $\intcc{-3}{-1}$ daalt kwadrateren: het grootste kwadraat komt van
$-3$: $1 \leq x^2 \leq 9$.

(c) Op $\intcc{-2}{3}$ bevat het \omterm{def:g10:numbers:interval}{interval} $0$, waar het kwadraat zijn
\omterm{def:g10:functions:extrema}{minimum} $0$ bereikt; het \omterm{def:g10:functions:extrema}{maximum} is het grotere van $(-2)^2 = 4$ en
$3^2 = 9$. Dus $0 \leq x^2 \leq 9$.
\end{solution}

\begin{solution}{exo:g10:reffunc:8}
Eerste bedrijf: $f(x) = 1.5x + 4$; tweede: $g(x) = 2x$. Het eerste is
goedkoper wanneer
\[
1.5x + 4 < 2x
\ \Longleftrightarrow\
4 < 0.5x
\ \Longleftrightarrow\
x > 8 .
\]
Voorbij $8$ km is het eerste bedrijf goedkoper; bij precies $8$ km rekenen
beide $16$.
\end{solution}

\begin{solution}{exo:g10:reffunc:9}
Beide kanten zijn niet-negatief, dus de ongelijkheid is equivalent met die
tussen hun kwadraten:
\[
\left(\sqrt{a+b}\right)^2 = a + b
\qquad\text{en}\qquad
\left(\sqrt a + \sqrt b\right)^2 = a + 2\sqrt a \sqrt b + b .
\]
Omdat $2\sqrt a\sqrt b \geq 0$, is het tweede kwadraat minstens het
eerste, wat $\sqrt{a+b} \leq \sqrt a + \sqrt b$ bewijst. Gelijkheid geldt
precies wanneer $\sqrt a \sqrt b = 0$, d.w.z.\ wanneer $a = 0$ of $b = 0$.
\end{solution}

\begin{solution}{exo:g10:reffunc:10}
\emph{1.} Zet het rechterlid op een gemeenschappelijke noemer:
\[
2 + \frac{3}{x-1} = \frac{2(x-1) + 3}{x - 1} = \frac{2x + 1}{x - 1}
= f(x).
\]

\emph{2.} Op $\intoo{1}{+\infty}$, als $x$ toeneemt, neemt $x - 1$ toe en
blijft positief, dus $\frac{3}{x-1}$ daalt (omgekeerde \omterm{def:g10:functions:function}{functie}, maal de
positieve constante $3$), dus $f(x) = 2 + \frac{3}{x-1}$ daalt: $f$ is
strikt \omterm{def:g10:functions:variations}{dalend} op $\intoo{1}{+\infty}$.
\end{solution}
